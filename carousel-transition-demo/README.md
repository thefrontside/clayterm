# transition carousel demo

A native Clay transitions demo for `clayterm`. The carousel keeps stable slide
IDs and lets Clay animate whole-frame enter/exit transitions.

## Run

```sh
cd .. && make
cd carousel-transition-demo && deno task demo
```

## Controls

- Click `Prev` and `Next`
- Use `Left` and `Right` arrow keys
- Press `q` or `Ctrl+C` to exit
