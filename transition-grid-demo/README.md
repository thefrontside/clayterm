# transition grid demo

A `clayterm` demo inspired by Clay's transition example. It shows a grid of
numbered rectangles that can be shuffled, recolored, and expanded using native
Clay transitions.

## Run

```sh
cd .. && make
cd transition-grid-demo && deno task demo
```

## Controls

- Click `Shuffle`, `Warm`, `Cool`, or `Expand`
- Press `s`, `w`, `c`, or `e`
- Press `q` or `Ctrl+C` to exit
