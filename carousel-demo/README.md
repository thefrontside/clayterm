# native carousel demo

A worktree-local demo that uses `clip.childOffset` to animate a horizontal
track inside a fixed 4:3 frame.

## Run

```sh
cd .. && make
cd carousel-demo && deno task demo
```

## Controls

- Click `Prev` and `Next`
- Use `Left` and `Right` arrow keys
- Press `q` or `Ctrl+C` to exit
