# floating carousel demo

A whole-frame carousel demo for `clayterm` that moves the bordered 4:3 frame as
one floating unit. The content stays attached to the frame and travels with it.

## Run

```sh
cd .. && make
cd carousel-floating-demo && deno task demo
```

## Controls

- Click `Prev` and `Next`
- Use `Left` and `Right` arrow keys
- Press `q` or `Ctrl+C` to exit
